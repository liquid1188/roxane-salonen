# Roxane B. Salonen — Peace Garden Passage
Author website designed and built by Lickity Split Web Design (Andrew Likoudis).

## Editing the site (for Roxane)
Visual editor at **/admin/** (Sveltia CMS). Sign in, edit with simple forms —
welcome text, featured book, photo album, speaking topics, testimonials.
Saving commits to `data/`; a GitHub Action renders the change into the pages
within ~1 minute. No code is ever shown or needed.

Auth: sign in with the personal access token provided by Lickity Split
(fine-grained, this repository only). An OAuth "Sign in with GitHub" worker
can replace this later.
